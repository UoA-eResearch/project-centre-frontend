# Prerequisites

- [bower](http://bower.io/)
- [sass](http://sass-lang.com/)

# Installation

```
bower install
```

# Configuration

## Create Stylesheet

```
cd app/styles
sass main.scss > main.css
```

## Set REST service base path

Adjust __REST_API_BASE_URL__ in app/scripts/app.js

The content of the app folder is the web application
